# Password Strength Checker

A client-side password strength checker that evaluates passwords in real time and provides visual feedback.

## Features

- **Real-time strength evaluation** — checks as you type
- **Visual strength meter** — color-coded bar from red (weak) to green (very strong)
- **Requirement checklist** — shows which criteria are met:
  - Minimum 8 characters
  - Uppercase letter
  - Lowercase letter
  - Number
  - Special character
- **Common password detection** — flags passwords like "password123" or "qwerty"
- **Password generator** — generates a random 16-character strong password
- **Copy to clipboard** — one-click copy for generated passwords
- **Show/hide toggle** — toggle password visibility

## Usage

Open `index.html` in any modern browser. No build tools or server required.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (no dependencies)
